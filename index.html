Minecraft Plugin

SimpleItemPrice

Real-time economy pricing for Minecraft. Shows live sell prices directly on items — everywhere, instantly, with no delay.

v1.0.0 Paper 1.21.11 EconomyShopGUI Bedrock Compatible
↓ Download Latest GitHub Repo →
Active — running live on LukeSMP

Core Features

💰

Live Price Display

Shows item sell value directly on items. Updates instantly when items move, split, stack, or transfer.

📦

Full Inventory Support

Works in player inventories, chests and containers, and EconomyShopGUI shops.

🧹

Smart GUI Filtering

Auto-hides prices on decorative GUI items like glass panes and nav buttons to keep menus clean.

🔗

Seamless Stacking

Items stack correctly regardless of price display. Old and new items normalized to prevent separation.

📱

Bedrock Compatible

Full Geyser/Floodgate support. Fixes drag splitting, double-tap stacking, and floating item bugs on mobile.

⚡

No Flicker / No Delay

Instant updates with zero visual glitches or inventory flickering.

Player Commands

/sip on Enable price display for yourself
/sip off Disable price display for yourself
/sip toggle Toggle price display on or off instantly

Compatibility

EconomyShopGUI (Free & Premium) ProtocolLib Geyser + Floodgate CommandPanels DeluxeMenus Paper 1.21.11

How to Install

Getting SimpleItemPrice running on your server takes under a minute.

1
Download the latest .jar from the releases page above.
2
Drop the file into your server's /plugins folder.
3
Restart your server. The plugin will generate its config automatically on first run.
