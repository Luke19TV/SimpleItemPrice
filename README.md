# SimpleItemPrice

> Real-time item sell price display for Paper 1.21.11, fully integrated with EconomyShopGUI. Live updates, and smart GUI filtering.

**Built by [LukeTV](https://luketv.net) — running live on LukeSMP**

---

## What is SimpleItemPrice?

SimpleItemPrice adds live sell price display directly onto items across your server. Players can instantly see what any item is worth — in their inventory, in chests, and in shops — without opening a menu or guessing. Fully integrated with EconomyShopGUI.

---

## Features

### 💰 Live Price Display
- Shows item sell value directly on items
- Updates instantly when items move, split, stack, or transfer

### 📦 Full Inventory Support
Works everywhere players interact with items:
- Player inventories
- Chests & containers
- EconomyShopGUI shops

### 🧹 Smart GUI Filtering
Automatically hides prices on:
- Decorative GUI items (glass panes, fillers)
- Navigation buttons (next page, paper icons, etc.)

Keeps your menus clean and professional.

### 🔗 Seamless Stacking
- Items stack correctly regardless of price display
- Old and new items are normalized to prevent separation issues

### ⚡ No Flicker / No Delay
- Instant updates with zero visual glitches
- No inventory flickering or input interruption

### 🛡️ Safe Packet Handling
- Prevents client disconnects and protocol errors
- Stable under heavy inventory interaction (shift-click, drag, crafting)

---

## Commands

| Command | Description |
|---|---|
| `/sip on` | Enable price display for yourself |
| `/sip off` | Disable price display for yourself |
| `/sip toggle` | Toggle price display on or off |
